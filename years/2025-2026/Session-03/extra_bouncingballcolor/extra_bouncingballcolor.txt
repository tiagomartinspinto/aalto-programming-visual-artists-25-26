OPTIONAL EXERCISE — BOUNCING BALL

Goal
Create a ball that moves across the screen and bounces off the edges.

What this reinforces:
• Variables
• Motion using +=
• Conditionals
• Logical OR (||)
• Changing values on collision

What to do
1) Copy/paste the sketch and run it.
2) Observe:
   - The ball moves diagonally.
   - It bounces when it hits the edges.
   - The color changes on each collision.

Play / Modify

A) Change speed
- Modify xSpeed and ySpeed.
- Try negative values.
- Try different values for each axis.

B) Change ball size
- Modify the radius variable.
- What breaks if the radius is very large?

C) Remove random colors
- Replace:
  ballColor = color(random(255), random(255), random(255));
  with a fixed color.

D) Add transparency
- Use:
  fill(ballColor, 150);

E) Add trail effect
- Comment out background(220);
- What happens?

F) Make it interactive
- On mousePressed(), reverse direction.
- On keyPressed(), increase speed.

Advanced Challenge
1) Add gravity (ySpeed += 0.1;)
2) Make multiple balls (hint: arrays)
3) Make speed depend on mouse position

Think about:
• What makes the ball bounce?
• What condition detects the edge?
• Why do we multiply speed by -1?

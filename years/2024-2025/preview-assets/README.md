# Preview Assets

Use this folder for screenshots and short GIFs of finished 2024-2025 sketches.

See the root [Preview Manifest](../../PREVIEW_MANIFEST.md) for suggested filenames and descriptions.

## Capture Checklist

1. Open the p5.js companion sketch in a browser.
2. Resize the window so the artwork is clear.
3. Capture a still screenshot for static sketches.
4. Capture a 4-8 second GIF or video for animated and interactive sketches.
5. Save the asset in this folder.
6. Link it from the relevant README or course page.

# Programming for Visual Artists

Aalto University course archive maintained by Tiago Martins Pinto.

This repository is organized by school year so course participants can find course materials, sketches, slides, examples, and course participant work in one place.

## School Years

- [2025-2026](years/2025-2026/)
- [2024-2025](years/2024-2025/)

## Structure

Each school year should stay self-contained:

```text
years/
  2025-2026/
  2024-2025/
```

When adding a new school year, create a new folder inside `years/`, add a year landing page, and link it from the root `index.html`.

# Programming for Visual Artists  
Aalto University

Programming for Visual Artists is a beginner-friendly creative coding course for arts students.
The course introduces programming as a tool for visual expression, interaction, and generative art.

Students learn the basics of coding through hands-on experiments with Processing and p5.js, focusing on play, exploration, and artistic thinking rather than technical perfection.

The course includes two short creative studies and a final project where students develop their own interactive or generative artwork.

No prior programming experience is required! :)

# Preview Assets

Use this folder for screenshots and short GIFs of finished sketches.

Suggested filenames:

- `session-01-face-exercise.png`
- `session-02-mouse-shapes.gif`
- `session-04-generative-grid.gif`

## Capture Checklist

1. Open the `.pde` sketch in Processing or open the p5.js companion sketch in a browser.
2. Resize the window so the artwork is clear.
3. Capture a still screenshot for static exercises.
4. Capture a 4-8 second GIF or video for animated and interactive exercises.
5. Save the asset in this folder.
6. Link it from `README.md` and `index.html`.

## Visual Guidelines

- Show the actual sketch output, not the Processing editor.
- Keep backgrounds and UI chrome out of the crop when possible.
- Prefer `png` for still images and `gif` or `mp4` for motion.
- Use lowercase names with hyphens.

The web sketches embedded in the course index act as live previews until exported image assets are added.

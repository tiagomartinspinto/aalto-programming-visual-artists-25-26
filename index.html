Web Sketches

These p5.js companions run directly in the page. Use them before downloading files when you want to preview behavior, compare syntax, or quickly experiment.

Web-first note

PDFs and `.pde` files remain available for the original lessons, but sketches that can be shown in-browser are embedded here. Open the web version first; download or clone the repository when you want to edit in Processing.

Face Exercise

A static drawing exercise translated from Processing to p5.js, built around a reusable `drawFace()` function.

Session 01functionsdrawing
Try: change face size, eye spacing, mouth arc, or call `drawFace()` more than once.
Open web sketchProcessing source

Mouse Shapes

Mouse position controls a circle, mirrored square, and changing background color.

Session 02mouseXmap()
Try: swap the mirrored movement, change the color mappings, or alter shape sizes.
Open web sketchProcessing source

Hover Grid

An 8 by 8 grid that changes the active cell when the pointer enters it.

Session 03conditionalsloops
Try: change rows and columns, use gradients, or make neighboring cells react too.
Open web sketchProcessing source

Bouncing Ball Color

A moving ball reverses direction at the canvas edges and changes color on each bounce.

Session 03motionstate
Try: adjust speed, radius, background fade, or add more bouncing objects.
Open web sketchProcessing source

Simple Waves

A sine wave drawn as a sequence of circles, animated over time.

Session 04sin()animation
Try: change amplitude, frequency, spacing, or draw connected lines instead of points.

Generative Grid

A rotating grid driven by Perlin noise, useful for seeing nested loops and smooth randomness.

Session 04noise()nested loops

Shapes Function

Session 05parametersrandomness

Spiral Spins

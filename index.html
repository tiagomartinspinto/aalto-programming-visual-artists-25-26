Processing vs p5.js

The web sketches are translations, not replacements. This quick map helps course participants move between the classroom Processing files and the browser versions.

Canvas Setup

Processing uses `size(600, 400)` inside `setup()`. p5.js uses `createCanvas(600, 400)` inside `function setup()`.

Drawing Loop

Processing writes `void draw()`. p5.js writes `function draw()`. Both redraw continuously unless stopped.

Transforms

Processing uses `pushMatrix()` and `popMatrix()`. p5.js shortens those to `push()` and `pop()`.

Variables

Processing often declares types such as `float x`. p5.js usually uses `let x`, `const x`, or class fields in JavaScript.
